## 🚀 Sobre o Projeto

Este repositório contém minhas atividades e desafios desenvolvidos durante a **Maratona de Ciência de Dados da Vivo**.

Ao longo da maratona, estou desenvolvendo habilidades essenciais para a área de Dados e Tecnologia, passando por conteúdos que envolvem lógica de programação, modelagem de banco de dados, programação em Python e criação de dashboards interativos para análise de dados.
